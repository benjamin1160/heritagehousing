# Landing Page Template

A single-focus, high-conversion landing page built on Next.js App Router + Tailwind. Every section is scaffolded to match a direct-response copywriting playbook:

1. Above-the-fold hero (avatar, benefit headline, subheadline, visual, primary CTA, credibility)
2. Problem section
3. Solution & offer with benefits + 3-step "How it works"
4. Social proof (stats, testimonials, logo wall) with an inline CTA
5. Risk reversal / guarantee near the CTAs
6. Objection-handling FAQ
7. Final conversion section with lean form and truthful urgency slot
8. Minimal header / footer — no distracting nav

## Filling in the copy

Every piece of business-specific content is marked with a visible, bracketed placeholder so you can find and replace them quickly:

```
[THIS IS A PLACEHOLDER FOR ...]
```

Search the project for `THIS IS A PLACEHOLDER` to get a full worklist.

Things you will need to gather before launch:

- The single primary conversion action (book call / demo / signup / purchase)
- Ideal customer avatar
- Dream-outcome headline with a specific number and timeframe
- Unique mechanism / differentiator for the subheadline
- Hero image or video of the "after" state
- 3 pain points in the customer's own words
- 3–7 benefits (not features)
- 3 simple steps of "how it works"
- Proof: 3+ testimonials with specific results, stats, client/media logos
- Guarantee / risk reversal
- 5–7 FAQ answers
- Truthful urgency or scarcity element (optional)
- Form handler endpoint
- Brand name, voice, colors, fonts
- Legal / compliance disclaimers

## How to Use

### One-Click Deploy

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/vercel/examples/tree/main/solutions/landing-page-template&project-name=landing-page-template&repository-name=landing-page-template)

### Clone and Run Locally

```bash
pnpm create next-app --example https://github.com/vercel/examples/tree/main/solutions/landing-page-template
pnpm dev
```

Then open http://localhost:3000 and start replacing placeholders.
